# Arduino 4 Servo Motors Sweep

## Project Overview
This project demonstrates controlling four servo motors using an Arduino Uno in Tinkercad.

The four servo motors perform a sweep motion for 2 seconds and then stop at 90°.

## Components
- Arduino Uno
- 4 × Micro Servo Motors
- Jumper Wires

## Circuit Connections
- Servo 1 → D3
- Servo 2 → D5
- Servo 3 → D6
- Servo 4 → D9
- All servos share 5V and GND.

## Project Function
- Controls four servo motors simultaneously.
- Performs a sweep motion for 2 seconds.
- Stops all servo motors at 90°.
